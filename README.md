## 📌 Sobre este repositório

Este repositório foi criado para documentar minha jornada de estudos na carreira de **Java pela Alura**.

Aqui vou registrar minha evolução, exercícios, anotações, códigos e pequenos projetos desenvolvidos durante os cursos.  
A ideia é manter um histórico real do meu aprendizado, mostrando prática, constância e progresso.

---

## 🎯 Objetivo

Meu objetivo com este repositório é fortalecer minha base em Java, lógica de programação e programação orientada a objetos, criando uma documentação organizada dos conteúdos estudados.

Este repositório também serve como parte do meu portfólio de estudos em tecnologia.

---

## 🧠 Conteúdos estudados

- Fundamentos da linguagem Java
- Variáveis, tipos de dados e operadores
- Estruturas condicionais
- Laços de repetição
- Leitura de dados com `Scanner`
- Programação Orientada a Objetos
- Classes, objetos, atributos e métodos
- Encapsulamento
- Construtores
- Boas práticas de organização de código
- Exercícios práticos
