# Java na Alura

Repositório criado para organizar meus estudos da carreira de **Java pela Alura**, registrando minha evolução, exercícios, anotações e projetos práticos.

O objetivo principal deste projeto é fortalecer minha base em **Java**, **lógica de programação**, **Programação Orientada a Objetos**, **entrada e saída de dados**, **estruturas condicionais**, **laços de repetição**, **arrays**, **métodos**, **classes**, **objetos** e boas práticas de desenvolvimento.

## Objetivo

Este repositório tem como objetivo registrar minha jornada de aprendizado em Java, acompanhando os cursos da Alura e praticando os principais fundamentos da linguagem.

A ideia não é apenas guardar códigos, mas também documentar meu progresso de forma organizada, mostrando minha evolução real como estudante de tecnologia.
